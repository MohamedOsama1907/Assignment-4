# Assignment-4 – Route Academy (📰 The UX Review Landing Page)

A fully responsive blog landing page built using HTML5 & modern CSS, recreating a real UX blog design without using any frameworks.

## 📌 Features

* Fully responsive design (Mobile → Tablet → Desktop)
* Built using Flexbox-based layout (as required)
* Sticky navigation bar with mobile menu (CSS-only interactivity)
* Clean and structured HTML layout
* Modern UI inspired by real blog platforms

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome (icons)
* Google Fonts

---

## 🎯 What I Practiced

* Using **Flexbox** to build layouts
* Creating responsive layouts with **media queries**
* Working with CSS positioning (relative, absolute, sticky)
* Building reusable components (cards, sections)
* Applying box model & box-sizing
* Using pseudo-elements (`::before`, `::after`)
* Creating hover effects and transitions
* Structuring a full multi-section landing page

---

## 📱 Responsive Breakpoints

* Mobile: Default
* Tablet: `min-width: 767px`
* Desktop: `min-width: 1024px`
* Large Screens: `min-width: 1370px`
